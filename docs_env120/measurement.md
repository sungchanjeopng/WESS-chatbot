# ENV120 (C1S-210) 측정 설정 가이드

## 측정 단위
| 설정 | 설명 |
|------|------|
| Meter (m) | 미터 (기본) |
| Feet (ft) | 피트 |

## 운영 모드
| 모드 | 설명 |
|------|------|
| Level | 슬러지 계면 레벨 측정 (기본) |
| Distance | 거리 측정 |

## 측정 파라미터
| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| Empty | 0~1000 (m) / 0~330 (ft) | 500 / 164 | 빈 구간 설정 |
| Dead Zone | 35~1000 (m) / 10~328 (ft) | 50 / 15 | 불감대 |
| Echo AMP | 1~50 | 15 | 에코 증폭 |
| Frequency | 130k/160k/270k/380k kHz | - | 초음파 주파수 |
| TX Interval | 1~60 | 3 | 송신 간격 |
| Offset | -100~+100 (m) / -32~+32 (ft) | 0 | 보정값 |
| Damping | 3초~60분 (9단계) | 3초 | 출력 안정화 |
| Threshold | Auto / Manual | Auto | 감지 임계값 |
| ASF | 0~9999 | 0 | 비정상 신호 필터 |
| Window Reset Time | 10~600 | 60 | 시간창 초기화 |
| Window Range | 10~100 | 20 | 시간창 범위 |

## 주파수 선택
| 주파수 | 특성 |
|--------|------|
| 380 kHz | 높은 분해능, 짧은 거리 (Light용) |
| 270 kHz | 중간 |
| 160 kHz | 긴 거리 (Heavy용) |
| 130 kHz | 가장 긴 거리 |

## 댐핑 단계
| 단계 | 시간 |
|------|------|
| 1 | 3초 (기본) |
| 2 | 30초 |
| 3 | 1분 |
| 4 | 2분 |
| 5 | 5분 |
| 6 | 10분 |
| 7 | 30분 |
| 8 | 60분 |

## Threshold (감지 임계값)
| 모드 | 범위 | 기본값 |
|------|------|--------|
| Auto (%) | 10~90% (5단위) | 50% |
| Manual (고정) | 20~220 (20단위) | 120 |

## 센서 세정
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 세정 간격 | 0~6000초 (60단위) | 60초 |
| 세정 시간 | 0~100초 | 3초 |
