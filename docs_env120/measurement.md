# ENV120 (C1S-210) 측정 설정 가이드

## 측정 단위
| 설정 | 설명 |
|------|------|
| Meter (m) | 미터 (기본) |
| Feet (ft) | 피트 |

## 운영 모드
| 모드 | 설명 |
|------|------|
| Level | 슬러지 계면 레벨 측정 (기본) |
| Distance | 거리 측정 |

| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| Empty | 0~1000 (m) / 0~330 (ft) | 500 / 164 | 빈 구간 설정 |
| Dead Zone | 35~1000 (m) / 10~328 (ft) | 50 / 15 | 불감대 |
| Echo AMP | 1~50 | 15 | 에코 증폭 |

| TX Interval | 1~60 | 3 | 송신 간격 |
| Offset | -100~+100 (m) / -32~+32 (ft) | 0 | 보정값 |

| Threshold | Auto / Manual | Auto | 감지 임계값 |

| 주파수 | 특성 |

| 모드 | 범위 | 기본값 |

## ENV120 수신 신호 조정 우선순위

- ENV120에서 에코 수신이 약하거나 측정이 불안정할 때는 문턱전압(Threshold)보다 수신감도(Echo AMP) 설정을 우선 확인/조정한다.
- Threshold/문턱전압은 수신감도, 센서 상태, 설치 조건, 에코 상태를 먼저 확인한 뒤 보조적으로 조정하는 항목으로 안내한다.
- 현장 답변에서는 먼저 센서 오염/설치 상태와 에코 강도를 확인하고, 그 다음 Echo AMP/수신감도, 마지막으로 Threshold/문턱전압 순서로 설명한다.

## ENV120 파형 화면 사진 판독 기준

- ENV120 파형 화면에서 상단 표시/선은 Empty 기준으로 판독한다.
- 하단 표시에서 `D`는 Distance/거리이고, `S`는 Sludge Level/슬러지 레벨이다.
- 사진 판독 답변에서는 `D`를 슬러지 레벨로, `S`를 거리로 혼동하지 않도록 명확히 구분한다.
