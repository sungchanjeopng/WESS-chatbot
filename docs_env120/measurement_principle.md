# ENV120 (C1S-210) 측정 원리 가이드

## 제품 정보
- 제품명: ENV120 초음파 슬러지 계면계
- 모델: C1S-210
- 펌웨어: v7.2.3
- 용도: 침전지, 농축조 등에서 슬러지 계면(경계층) 위치를 실시간 측정

## 측정 원리
ENV120은 초음파 반사 방식으로 슬러지 계면을 측정합니다.

### 동작 과정
1. 센서에서 초음파 펄스를 수면 아래로 발사
2. 초음파가 슬러지 층을 만나면 반사
3. 반사된 에코 신호를 수신 (200개 샘플 분석)
4. 에코 신호의 도달 시간과 음속으로 거리 계산
5. 거리에서 계면 레벨(깊이)을 산출

### 측정값 종류
| 값 | 설명 | 단위 |
|----|------|------|
| Level | 슬러지 계면 깊이 (수면 기준) | m 또는 ft |
| Distance | 센서에서 계면까지 거리 | m 또는 ft |

## 신호 처리 과정

```
초음파 발사 → 에코 수신 → 에코 감지 → Threshold 판별 → 추적(Tracking) → 댐핑 → 최종 레벨
```

### 1단계: 에코 감지 (Echo Check)
- 수신 신호에서 슬러지 계면 에코를 찾습니다
- 에코 강도(Echo AMP)를 증폭하여 감도 조절
- 설정 범위: 1~50 (기본: 15)

### 2단계: Threshold (임계값) 판별
- 에코 신호에서 계면 위치를 결정하는 기준값
- **Auto 모드**: 신호 크기에 비례하여 자동 설정 (10~90%, 기본 50%)
- **Manual 모드**: 고정 임계값 직접 설정 (20~220, 기본 120)

### 3단계: ASF (Abnormal Signal Filter)
- 센서 근접 영역의 비정상 반사 신호 제거
- 설정 범위: 0~9999 (기본: 0)

### 4단계: 시간창 추적 (Time Window Tracking)
- 이전 측정 위치 근처에서만 에코를 탐색하여 안정성 향상
- Window Reset Time: 10~600초 (기본: 60초) - 에코 소실 시 전체 범위로 복귀하는 시간
- Window Range: 10~100 (기본: 20) - 탐색 범위 폭
- 에코가 시간창 밖에서 감지되면 일정 시간 후 창을 리셋

### 5단계: 댐핑 (Damping)
- 이동평균 필터로 출력값 안정화
- 9단계 설정: 3초 / 30초 / 1분 / 2분 / 5분 / 10분 / 30분 / 60분
- 기본값: 3초
- 값이 클수록 안정적이지만 응답 느림

### 6단계: 오프셋 (Offset)
- 최종 측정값에 보정값 추가
- 범위: -100~+100 cm (미터) / -32~+32 (피트)
- 기본값: 0

## 주파수 선택
| 주파수 | 특성 | 권장 용도 |
|--------|------|-----------|
| 380 kHz | 높은 분해능, 짧은 도달 거리 | 얕은 침전지 |
| 270 kHz | 중간 | 일반 침전지 |
| 160 kHz | 낮은 분해능, 긴 도달 거리 | 깊은 침전지 |
| 130 kHz | 가장 긴 도달 거리 | 매우 깊은 침전지 |

## Dead Zone (불감대)
- 센서 전면의 측정 불가 영역
- 미터: 35~1000 (기본: 50)
- 피트: 10~328 (기본: 15)
- 불감대 내의 에코는 무시됨

## Empty (빈 구간)
- 슬러지가 없는 것으로 판단하는 최대 거리
- 미터: 0~1000 (기본: 500)
- 피트: 0~330 (기본: 164)
- 이 값을 초과하면 빈 상태로 판정

## 에러 종류
| 에러 | 설명 | 조치 |
|------|------|------|
| Echo Error | 에코 신호 감지 안 됨 | 센서 확인, 측정 대상 확인 |
| Temperature Error | 온도 센서 이상 | 온도 센서 연결 확인 |

## 온도 보정
- 내장 온도 센서로 수온 실시간 측정
- 수온에 따라 음속(Velocity)이 변하므로 자동 보정 적용
- 음속 변화를 반영하여 거리 계산 정확도 유지

## 단위 변환
- 미터 → 피트: × 0.3281
- 피트 → 미터: × 3.048

## 센서 세정 (Probe Clean Device)
- 센서 표면 오염 시 자동 세정 기능
- 세정 간격: 0~6000초 (기본: 60초, 60초 단위)
- 세정 시간: 0~100초 (기본: 3초)
- 세정 중에는 측정이 일시 중단됨
