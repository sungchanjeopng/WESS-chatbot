# ENV120 (C1S-210) 출력 설정 가이드

## 4-20mA 전류 출력
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 4mA 설정 | 0~1000 (m) / 0~328 (ft) | 0 |
| 20mA 설정 | 0~1000 (m) / 0~328 (ft) | 500 |
| 12mA 트림 | 0~99 | 50 |
| 20mA 트림 | 0~99 | 50 |

## 에러 출력
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 에러 지연 | 20~990초 (10단위) | 120초 |
| 에러 출력 | 3.8mA / Hold / 21mA | 3.8mA |

## 릴레이
릴레이 2채널 지원 (R1, R2).

### 릴레이 1
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 동작값 (Act) | 0~1000 | 900 |
| 정지값 (Stop) | 0~1000 | 800 |

### 릴레이 2
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 동작값 (Act) | 0~1000 | 0 |
| 정지값 (Stop) | 0~1000 | 10 |

## 통신 프로토콜
| 설정 | 설명 |
|------|------|
| Standard | 일반 시리얼 |
| RF | RF 무선 통신 |
| Modbus | Modbus RTU |

| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 보레이트 | 4800 / 9600 / 19200 | - |
| 슬레이브 주소 | 1~247 | 1 |

### RF 설정
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| RF 채널 | 1~4 | 1 |
