# ENV130 (C1D-330) 측정 설정 가이드

## 제품 정보
- 제품명: ENV130 초음파 슬러지 계면계
- 모델: C1D-330
- 펌웨어: v1.0.9

## 측정 단위
| 설정 | 설명 |
|------|------|
| Meter (m) | 미터 단위 (기본값) |

## 채널 운영 모드
듀얼 채널(CH1/CH2) 또는 싱글 채널 운영 가능.

### 운영 모드 (Operation)
| 모드 | 설명 |
|------|------|
| Sludge | 슬러지 계면 측정 (기본값) |
| Distance | 거리 측정 |

### Empty Pipe (빈 구간 설정)
- 설정 범위: 1 ~ 1000
- 기본값: 500

### Dead Zone (불감대)
- 설정 범위: 35 ~ 1000
- 기본값: 50

### 측정 주기 (Measure Cycle)
- 설정 범위: 1 ~ 50
- 기본값: 3

### 측정 초기화 (Measure Reset)
- No / Yes 선택

## 교정 (Calibration) 파라미터

### 주파수 (Frequency)
| 설정 | 주파수 |
|------|--------|
| 0 | 380 kHz (기본값) |
| 1 | 270 kHz |
| 2 | 160 kHz |
| 3 | 130 kHz |

### Echo AMP (에코 증폭)
- 설정 범위: 1 ~ 50
- 기본값: 15

### Threshold (감지 임계값)
| 모드 | 설명 |
|------|------|
| Auto | 자동 임계값 (기본값) |
| Manual | 수동 설정 |

- 자동 모드 범위: 0 ~ 95 (기본값: 50)
- 수동 모드 범위: 0 ~ 32 (기본값: 16)
- Heavy/Light 각각 설정 가능

### ASF (Auto Sensitivity Function)
- 설정 범위: 0 ~ 9999
- 기본값: 0
- Heavy/Light 채널별 설정

### 댐핑 (Damping)
- 설정 범위: 1 ~ 100
- 기본값: 20
- 값이 클수록 안정적, 응답 느림

### 오프셋 (Offset)
- 설정 범위: -100 ~ +100
- 기본값: 0
