# ENV130 (C1D-330) 시스템 설정 가이드

## 시스템 메뉴

### 비밀번호 (Passcode)
- 설정 범위: 0 ~ 9999
- 기본값: 0 (비밀번호 없음)

### 언어 (Language)
| 설정 | 언어 |
|------|------|
| English | 영어 (기본값) |
| Korean | 한국어 |

### 시간 설정 (Time)
- 년/월/일/시/분 설정

### 사이트 이름 (Site Name)
- CH1/CH2 각각 설정 가능
- 형식: 영문 대문자 1자 + 숫자 2자리 (예: A01)
- 영문: A~Z, 숫자: 01~99

### 공장 초기화 (Factory Reset)
- No / Yes 선택
- 모든 설정값을 공장 기본값으로 복원
