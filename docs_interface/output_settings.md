# ENV130 (C1D-330) 출력 설정 가이드

## 4-20mA 전류 출력

### 채널 구성
듀얼 채널(CH1/CH2) 각각 독립 설정 가능.

### 출력 할당 (Assign)
| 설정 | 설명 |
|------|------|
| Light | 옅은 층 계면값 출력 |
| Heavy | 진한 층 계면값 출력 (기본값) |

### 4mA / 20mA 설정값
- 4mA 설정: 0 ~ 1000 (기본값: 0)
- 20mA 설정: 0 ~ 1000 (기본값: 500)

### 트림 (미세 조정)
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 12mA 트림 | -100 ~ +100 | 0 |
| 20mA 트림 | -100 ~ +100 | 0 |

### 4mA 수동 출력
메뉴에서 4mA 수동 출력하여 루프 테스트 가능.

## 릴레이

### 릴레이 할당 (Assign)
| 설정 | 설명 |
|------|------|
| CH1 Light | 채널1 옅은 층 |
| CH1 Heavy | 채널1 진한 층 (기본값) |
| CH2 Light | 채널2 옅은 층 |
| CH2 Heavy | 채널2 진한 층 |

### 릴레이 동작/정지
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 동작값 (Act) | 0 ~ 1000 | 900 |
| 정지값 (Stop) | 0 ~ 1000 | 800 |

### 릴레이 테스트
메뉴에서 릴레이 수동 ON/OFF 테스트 가능. (범위: 0 ~ 1000)

## 센서 세정 (Clean)

### 세정 모드
- 설정 범위: 0 ~ 2
- 기본값: 0 (비활성)

### 세정 간격 (Interval)
- 설정 범위: 1 ~ 6000초
- 기본값: 60초

### 세정 시간 (Term)
- 설정 범위: 1 ~ 100초
- 기본값: 3초

## 에러 출력

### 에러 지연 시간
- 설정 범위: 20 ~ 999초
- 기본값: 120초

### 에러 시 출력 동작
| 설정 | 동작 |
|------|------|
| Hold | 에러 직전 출력값 유지 (기본값) |
| 3.8mA | 하한 경보 전류 |
| 21mA | 상한 경보 전류 |
