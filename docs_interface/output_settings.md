# ENV130 (C1D-330) 출력 설정 가이드

## 4-20mA 전류 출력
- 할당: Light(옅은 층) / Heavy(진한 층, 기본)
- 4mA 설정: 0~1000 (기본: 0)
- 20mA 설정: 0~1000 (기본: 500)
- 트림: -100~+100 (기본: 0)

## 릴레이
- 할당: CH1 Light/Heavy, CH2 Light/Heavy
- 동작값: 0~1000 (기본: 900)
- 정지값: 0~1000 (기본: 800)

## 센서 세정
- 세정 간격: 1~6000초 (기본: 60초)
- 세정 시간: 1~100초 (기본: 3초)

## 에러 출력
- 에러 지연: 20~999초 (기본: 120초)
- 출력 모드: Hold / 3.8mA / 21mA (기본: Hold)
