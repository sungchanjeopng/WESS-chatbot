# ENV130 (C1D-330) 데이터 및 통신 설정 가이드

## 데이터 저장 (Trend)

### 저장 간격 (Interval)
| 설정 | 간격 |
|------|------|
| 1초 | 1초마다 저장 |
| 10초 | 10초마다 저장 |
| 1분 | 1분마다 저장 (기본값) |
| 10분 | 10분마다 저장 |
| 1시간 | 1시간마다 저장 |

### 데이터 다운로드
- 시리얼 포트를 통해 저장된 데이터 다운로드

### 데이터 삭제
- No / Yes 선택
- 삭제 시 복구 불가

### 디스플레이 기간
| 설정 | 표시 기간 |
|------|-----------|
| Term 1 | 기간 1 |
| Term 2 | 기간 2 |

## 파형 저장 (Echo Save)

### 저장 모드
| 모드 | 설명 |
|------|------|
| Real | 실시간 파형 저장 |
| Average | 평균 파형 저장 (기본값) |

### 파형 저장 간격
| 설정 | 간격 |
|------|------|
| 1초 | 1초마다 |
| 10초 | 10초마다 |
| 1분 | 1분마다 (기본값) |
| 10분 | 10분마다 |
| 1시간 | 1시간마다 |

### 파형 보기 모드
| 모드 | 설명 |
|------|------|
| 측정 (MSR) | 측정 파형 보기 (기본값) |
| 자동교정 (Auto Cal) | 자동교정 파형 보기 |

## 통신 설정 (Communication)

### 통신 타입
| 설정 | 설명 |
|------|------|
| Modbus | Modbus RTU 통신 (기본값) |
| RF | RF 무선 통신 |

### Modbus 설정
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 보레이트 | 9600 / 19200 / 115200 | 19200 |
| 슬레이브 주소 | 1 ~ 255 | 1 |

### RF 통신 설정
| 파라미터 | 설명 |
|----------|------|
| 할당 (Assign) | Light / Heavy (기본: Heavy) |
| 주소 (Address) | 1 ~ 4 (기본: 1) |
- CH1/CH2 각각 설정 가능

## 메뉴 구조 (공개 섹션만)

```
메인 메뉴
├── 1. 측정 (Measurement)
│   ├── Base (단위, 운영모드, Empty, 불감대, 측정주기)
│   └── Calibration (주파수, Echo AMP, Threshold, ASF, 댐핑, 오프셋)
├── 2. 출력 (Output)
│   ├── Current (4-20mA 설정)
│   ├── Relay (릴레이 설정)
│   ├── Clean (센서 세정)
│   └── Error (에러 출력)
├── 3. 데이터 (Data)
│   ├── Trend (데이터 저장/다운로드/삭제)
│   ├── Communication (Modbus/RF 설정)
│   └── Echo (파형 저장)
└── 4. 시스템 (System)
    ├── Passcode (비밀번호)
    ├── Language (언어)
    ├── Time (시간 설정)
    ├── Site Name (사이트 이름)
    └── Factory Reset (공장 초기화)
```
