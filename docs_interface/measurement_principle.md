# ENV130 (C1D-330) 측정 원리 가이드

## 제품 정보
- 제품명: ENV130 초음파 슬러지 계면계
- 모델: C1D-330
- 용도: 하수처리장, 정수장 등 침전지에서 슬러지 계면(경계층) 위치를 실시간 측정

## 측정 원리

ENV130은 **초음파 반사 방식**으로 슬러지 계면을 측정합니다.

### 동작 과정
1. 센서에서 초음파 펄스를 수면 아래로 발사
2. 초음파가 슬러지 층을 만나면 반사
3. 반사된 에코 신호를 수신
4. 에코 신호의 위치(시간)로 계면 깊이를 계산
5. 옅은 층(Light)과 진한 층(Heavy) 두 계면을 동시에 감지

### 듀얼 계면 측정
ENV130은 하나의 센서로 **두 개의 계면**을 동시에 측정합니다:
- **Light (옅은 층)**: 상부 슬러지 경계면 (맑은 물과 옅은 슬러지 경계)
- **Heavy (진한 층)**: 하부 슬러지 경계면 (옅은 슬러지와 진한 슬러지 경계)

이를 통해 슬러지 블랭킷의 두께와 위치를 정확히 파악할 수 있습니다.

## 채널 구성

### 듀얼 채널
- CH1, CH2 두 개의 독립 채널 운영
- 각 채널별 독립적인 측정/교정/출력 설정
- 센서 2개로 서로 다른 위치 동시 측정

### 싱글 채널
- CH1만 사용하는 단일 채널 운영
- 센서 1개로 운영

## 신호 처리 과정

```
초음파 발사 → 에코 수신 → 프로파일 축적 → Threshold 감지 → ASF 적용 → 댐핑 → 최종 결과
```

### 1단계: 프로파일 축적 (Profile)
- 여러 번의 에코 신호를 축적하여 노이즈 감소
- 최대 100회 프로파일 축적
- 축적 횟수가 많을수록 안정적이나 응답 느림

### 2단계: Threshold (임계값) 감지
- 에코 신호에서 슬러지 계면 위치를 찾는 기준값
- **Auto 모드**: 신호의 최대/최소값을 기반으로 자동 설정 (기본값)
- **Manual 모드**: 사용자가 직접 임계값 설정
- Light(옅은 층)과 Heavy(진한 층) 각각 별도 Threshold 설정

### 3단계: ASF (Auto Sensitivity Function)
- 센서 근접 영역의 잔여 반사 신호를 자동 보정
- Light/Heavy 각각 설정 가능
- 설정 범위: 0 ~ 9999

### 4단계: 댐핑 (Damping)
- 이동평균 필터로 출력값 안정화
- 설정 범위: 1 ~ 100 (기본값: 20)
- 값이 클수록 안정적이지만 응답 느림

### 5단계: 오프셋 (Offset)
- 최종 측정값에 보정값 추가
- 설정 범위: -100 ~ +100 (기본값: 0)

## 주파수 선택

| 주파수 | 특성 | 권장 용도 |
|--------|------|-----------|
| 380 kHz (기본) | 높은 분해능, 짧은 도달 거리 | 일반 침전지 |
| 270 kHz | 중간 성능 | 중간 깊이 |
| 160 kHz | 낮은 분해능, 긴 도달 거리 | 깊은 침전지 |
| 130 kHz | 가장 긴 도달 거리 | 매우 깊은 침전지 |

## 측정 모드

| 모드 | 설명 |
|------|------|
| Sludge (슬러지) | 슬러지 계면 깊이를 m 단위로 표시 (기본값) |
| Distance (거리) | 센서에서 계면까지 거리를 표시 |

## Dead Zone (불감대)
- 센서 전면의 측정 불가 영역
- 설정 범위: 35 ~ 1000 (기본값: 50)
- 불감대 내의 에코 신호는 무시됨

## Empty (빈 구간)
- 슬러지가 없는 것으로 판단하는 기준값
- 설정 범위: 1 ~ 1000 (기본값: 500)

## 신호 안정화
- 신호가 정상(OK)으로 판정되려면 15초간 안정적이어야 함
- 신호가 비정상(NG)으로 판정되려면 15초간 지속되어야 함
- 일시적인 신호 변동에 의한 오측정 방지

## 자동 게인 (Auto Gain)
- 수신 감도를 자동으로 최적화하는 기능
- 채널별 독립 설정
- 최적의 에코 신호 강도를 자동으로 찾아 설정

## 자동 교정 (Auto Calibration)
- 센서와 측정 환경에 맞게 자동으로 교정
- 최적의 에코 증폭, Threshold, 기울기(slope) 값을 자동 탐색
- 교정 결과: Echo AMP, Slope, Threshold, 전압값 등

## 온도 보정
- 수온에 따라 초음파 속도가 변하므로 온도 보정 적용
- 내장 온도 센서로 실시간 수온 측정
- 온도에 따른 음속 보정을 자동으로 계산하여 측정 정확도 유지
