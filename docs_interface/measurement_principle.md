# ENV130 (C1D-330) 측정 원리 가이드

## 측정 원리
ENV130은 초음파 반사 방식으로 슬러지 계면을 측정합니다.

### 동작 과정
1. 센서에서 초음파 펄스를 수면 아래로 발사
2. 슬러지 층에서 반사된 에코 신호 수신
3. 에코 위치로 계면 깊이 계산
4. Light(옅은 층)과 Heavy(진한 층) 두 계면 동시 감지

### 신호 처리
프로파일 축적 → Threshold 감지 → ASF 적용 → 댐핑 → 최종 결과

### 주파수 선택
| 주파수 | 특성 |
|--------|------|
| 380 kHz | 높은 분해능, 짧은 거리 (기본) |
| 270 kHz | 중간 |
| 160 kHz | 낮은 분해능, 긴 거리 |
| 130 kHz | 가장 긴 거리 |
