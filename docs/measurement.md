# ENV200 (C2S-211) 측정 원리 및 설정 가이드

## 측정 원리
ENV200은 초음파 감쇄(Ultrasonic Attenuation) 방식으로 슬러지 농도를 측정합니다.
1. 센서에서 초음파 펄스를 발사
2. 슬러지를 통과하며 감쇄된 신호를 수신
3. 에너지평균법(EEAM)으로 감쇄율 분석
4. 교정 데이터를 적용하여 농도값 산출

## 농도 단위
| 단위 | 설명 |
|------|------|
| % | 백분율 (기본) |
| ppm | 백만분율 (1% = 10,000 ppm) |
| mg/L | 밀리그램/리터 (= ppm) |
| g/L | 그램/리터 (1g/L = 0.1%) |
| kg/m3 | 플랜트 모드 전용 |
| g/cm3 | 플랜트 모드 전용 |

## 배관 외경 설정
센서 설치 배관의 외경에 따라 3가지 범위를 선택합니다:
| 설정 | 배관 외경 범위 |
|------|---------------|
| 0~200mm | 소구경 배관 |
| 200~400mm | 중구경 배관 |
| 400~600mm | 대구경 배관 |

## 검출 범위 (Detection Area)
- 초음파 신호에서 농도를 분석하는 영역을 설정
- Low: 1 (기본), High: 229 (기본)
- 범위: 1~229

## 초음파 주파수
- 자동 모드: 최적 주파수를 자동으로 탐색 (기본)
- 수동 모드: 1~47 범위에서 직접 설정 (기본: 21)
- 자동 모드에서는 5사이클마다 최적 주파수를 재탐색

## 댐핑 (Damping)
- 출력값을 안정화하는 이동평균 필터
- 설정 범위: 1~900
- 기본값: 60
- 값이 클수록: 안정적이지만 응답 느림
- 값이 작을수록: 응답 빠르지만 변동 큼

### 권장값
| 상황 | 댐핑값 |
|------|--------|
| 빠른 응답 필요 | 1~10 |
| 일반 운전 | 30~100 |
| 매우 안정적 출력 | 200~900 |

## 오프셋 (Offset)
- 측정값에 일정 보정값을 더합니다
- 일반 모드: -100 ~ +100
- 플랜트 모드: -1000 ~ +1000
- 기본값: 0

## 최대 측정 범위
- 측정 가능한 최대 농도값 설정
- 일반 모드: 1~9000 (기본: 800)
- 플랜트 모드: 1~9999 (기본: 2000)

## 온도 측정
- 내장 온도 센서로 수온을 실시간 측정
- 온도에 따른 음속 보정 자동 적용
- 온도 에러 시 에러 출력 동작 수행
