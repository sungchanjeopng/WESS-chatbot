# ENV200 (C2S-211) 측정 원리 및 설정 가이드

## 측정 원리
ENV200은 초음파 감쇄(Ultrasonic Attenuation) 방식으로 슬러지 농도를 측정합니다.
1. 센서에서 초음파 펄스를 발사
2. 슬러지를 통과하며 감쇄된 신호를 수신
3. 에너지평균법(EEAM)으로 감쇄율 분석
4. 교정 데이터를 적용하여 농도값 산출

## 농도 단위
| 단위 | 설명 |
|------|------|
| % | 백분율 (기본) |
| ppm | 백만분율 |
| mg/L | 밀리그램/리터 |
| g/L | 그램/리터 |

## 배관 외경 설정
| 설정 | 배관 외경 범위 |
|------|---------------|
| 0~200mm | 소구경 배관 |
| 200~400mm | 중구경 배관 |
| 400~600mm | 대구경 배관 |

## 초음파 주파수
- 자동 모드: 최적 주파수를 자동으로 탐색 (기본)
- 수동 모드: 1~47 범위에서 직접 설정 (기본: 21)

## 댐핑 (Damping)
- 설정 범위: 1~900, 기본값: 60
- 값이 클수록 안정적이지만 응답 느림

## 오프셋 (Offset)
- 일반 모드: -100 ~ +100
- 플랜트 모드: -1000 ~ +1000, 기본값: 0

## 온도 측정
- 내장 온도 센서로 수온을 실시간 측정
- 온도에 따른 음속 보정 자동 적용
