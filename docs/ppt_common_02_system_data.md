# 공통 시스템 및 데이터 섹션 가이드

## 1. 시스템 (System)

### Passcode (비밀번호)
- 메뉴 진입 시 비밀번호 입력 필요
- 범위: 0~9999
- 기본값: 0 (비밀번호 없음)

### Language (언어)
- English (영어)
- Korean (한국어)
- Chinese (중국어)
- Japanese (일본어)

### Time (시간 설정)
- 년/월/일/시/분 설정
- RTC(실시간 시계)로 시간 유지

### Version (버전)
- 현재 펌웨어 버전 확인

## 2. 데이터 (Data)

### Save (저장)
- 데이터 저장 ON/OFF 설정

### Saving Interval (저장 간격)
- 1분 / 10분 / 60분 선택

### Download (다운로드)
- 시리얼 포트를 통해 저장된 데이터 다운로드

### Delete (삭제)
- 저장된 데이터 전체 삭제
- 삭제 후 복구 불가

### Display Term (표시 기간)
- 트렌드 화면에 표시할 데이터 기간 설정
