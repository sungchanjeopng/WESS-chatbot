# ENV200 (C2S-211) 에러 코드 가이드

## 에러 종류

| 에러 | 설명 | 조치 방법 |
|------|------|-----------|
| **신호 없음 (No Signal)** | 초음파 수신 신호가 감지되지 않음 | 센서 연결 상태 확인, 센서 표면 오염/기포 제거, 측정 대상 확인 |
| **온도 에러 (Temperature)** | 온도 센서 이상 | 온도 센서 연결 확인, 온도 센서 교체 필요 시 A/S 연락 |

## 에러 발생 시 동작

### 4-20mA 출력
에러 발생 시 전류 출력 동작을 선택할 수 있습니다:
| 설정 | 동작 |
|------|------|
| 유지 (Hold) | 에러 발생 직전의 출력값 유지 |
| 3.8mA | 하한 경보 전류 출력 |
| 21mA | 상한 경보 전류 출력 |

### 에러 지연 시간
- 에러 조건이 지속되어야 에러로 판정되는 시간
- 설정 범위: 20 ~ 999초
- 기본값: 120초
- 일시적인 신호 불안정으로 인한 오경보 방지

### 에러 상태 확인
- LCD 메인 화면 상단에 에러 코드 표시
- Modbus 통신으로 원격 확인 가능
