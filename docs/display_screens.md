# ENV200 (C2S-211) 화면 구성 가이드

## 디스플레이 사양
| 항목 | 사양 |
|------|------|
| 디스플레이 종류 | LCD (그래픽) |
| 언어 | 한국어 / 중국어 / 일본어 |

## 화면 종류

### 시작 화면
전원 투입 시 표시됩니다.
- 제품명: ENV200
- 모델: C2S-211
- 펌웨어 버전: V8.0.3

### 메인 측정 화면
정상 운전 시 기본 화면입니다.
- 현재 농도값 (대형 숫자)
- 농도 단위 (%, ppm, mg/L, g/L 등)
- 에러 상태 표시
- 릴레이 동작 상태

### Echo 화면
초음파 수신 파형을 실시간으로 표시합니다.
- 초음파 에코 파형
- 검출 범위 (Detection Area) 표시
- 신호 분석에 활용

### 메뉴 화면
ENTER 버튼으로 메뉴에 진입합니다.
- 시스템 / 측정 / 출력 / 데이터 4개 섹션
- UP/DOWN으로 항목 이동
- ENTER로 선택, ESC로 뒤로가기

### 교정 화면
교정 진행 시 표시됩니다.
- 교정 진행 상태
- 현재 측정값

## 버튼 조작
| 버튼 | 기능 |
|------|------|
| ENTER | 메뉴 진입 / 항목 선택 / 값 확인 |
| ESC | 취소 / 이전 화면 |
| UP | 값 증가 / 이전 항목 |
| DOWN | 값 감소 / 다음 항목 |
