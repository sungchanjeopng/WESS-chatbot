# ENV200 (C2S-211) 화면 구성 가이드

## 화면 종류

### 시작 화면
전원 투입 시 표시. 제품명, 모델, 펌웨어 버전.

### 메인 측정 화면
- 현재 농도값 (대형 숫자)
- 농도 단위
- 에러 상태
- 릴레이 동작 상태

### Echo 화면
초음파 수신 파형을 실시간 표시. 검출 범위(Detection Area) 표시.

### 메뉴 화면
ENTER 버튼으로 진입. 시스템/측정/출력/데이터 4개 섹션.

## 버튼 조작
| 버튼 | 기능 |
|------|------|
| ENTER | 메뉴 진입 / 항목 선택 / 값 확인 |
| ESC | 취소 / 이전 화면 |
| UP | 값 증가 / 이전 항목 |
| DOWN | 값 감소 / 다음 항목 |
