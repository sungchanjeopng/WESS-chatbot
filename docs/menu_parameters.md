# ENV200 (C2S-211) 메뉴 구조 및 파라미터

## 메뉴 구조

```
메인 메뉴
├── 1. 시스템 (System)
├── 2. 측정 (Measure)
├── 3. 출력 (Output)
└── 4. 데이터 (Data)
```

## 1. 시스템 메뉴
- 언어 설정 (한국어/중국어/일본어)
- 비밀번호 설정
- 공장 초기화

## 2. 측정 메뉴

| 파라미터 | 설명 | 범위 | 기본값 |
|----------|------|------|--------|
| 농도 단위 | 측정 단위 선택 | %, ppm, mg/L, g/L | % |
| 배관 외경 | 센서 설치 배관의 외경 | 0~200, 200~400, 400~600mm | 0~200mm |
| 검출 범위 | 측정 감지 영역 (Low/High) | 1~229 | Low:1, High:229 |
| 주파수 | 초음파 주파수 설정 | 1~47 (자동/수동) | 자동, 21 |
| 교정 | 2점/다점 교정 실행 | - | - |
| 댐핑 | 출력 안정화 (이동평균 횟수) | 1~900 | 60 |
| 오프셋 | 농도 보정값 | -100~+100 (일반), -1000~+1000 (플랜트) | 0 |
| 최대 범위 | 측정 최대 범위 | 1~9000 | 800 |
| 교정 초기화 | 교정 데이터 삭제 | - | - |

## 3. 출력 메뉴

| 파라미터 | 설명 | 범위 | 기본값 |
|----------|------|------|--------|
| 4mA 설정값 | 4mA에 해당하는 농도값 | 0~9999 | 0 (일반), 1000 (플랜트) |
| 20mA 설정값 | 20mA에 해당하는 농도값 | 0~9999 | 800 (일반), 1200 (플랜트) |
| 12mA 트림 | 중간점 미세 조정 | 0~99 | 50 |
| 20mA 트림 | 상한점 미세 조정 | 0~99 | 50 |
| 4mA 출력 | 4mA 수동 출력 테스트 | - | - |
| 에러 지연 | 에러 판정 대기 시간 | 20~999초 | 120초 |
| 에러 출력 | 에러 시 전류 출력 모드 | 유지/3.8mA/21mA | 유지 |
| 릴레이 동작값 | 릴레이 ON 기준 농도 | - | 800 (일반), 1150 (플랜트) |
| 릴레이 정지값 | 릴레이 OFF 기준 농도 | - | 700 (일반), 1140 (플랜트) |
| 릴레이 테스트 | 릴레이 수동 ON/OFF | - | - |
| 프로토콜 | 통신 방식 선택 | Standard/Modbus | Modbus |
| 외부 입력 1/2 | 외부 입력 신호 설정 | ON/OFF | OFF |
| 외부 출력 | 외부 입력 시 출력 동작 | 유지/4mA | 유지 |

## 4. 데이터 메뉴
- 저장 간격 설정
- 데이터 보기
- 데이터 다운로드
- 데이터 삭제

## 플랜트 모드 vs 일반 모드
제품은 **플랜트(Plant)** 모드와 **일반(General)** 모드 두 가지로 운영됩니다.
- **플랜트**: 정수장/하수처리장 전용 (단위: kg/m3 또는 g/cm3)
- **일반**: 범용 (단위: %, ppm, mg/L, g/L)

모드에 따라 일부 파라미터의 범위와 기본값이 달라집니다.
