# ENV200 (C2S-211) 교정(Calibration) 가이드

## 교정 개요
정확한 농도 측정을 위해서는 실제 현장 조건에 맞는 교정이 필요합니다. 알려진 농도의 기준 시료를 사용하여 센서의 원시 측정값과 실제 농도 간의 관계를 보정합니다.

## 교정 방법

### 2점 교정
1. 메뉴 > 측정 > 교정 진입
2. 낮은 농도 기준 시료로 영점(Zero) 교정
3. 높은 농도 기준 시료로 스팬(Span) 교정
4. 두 점 사이를 직선 보간하여 농도 산출

### 다점 교정
1. 메뉴 > 측정 > 교정 진입
2. 여러 개의 기준 시료로 교정점 추가
3. 각 구간 사이를 선형 보간
4. 넓은 범위이거나 비선형 관계일 때 적합

## 교정 절차
1. 기준 시료를 준비합니다 (실험실 분석값 권장)
2. 센서를 기준 시료에 담급니다
3. 메뉴에서 교정을 시작합니다
4. 기준 시료의 실제 농도값을 입력합니다
5. 안정화 대기 후 자동으로 교정이 수행됩니다

## 교정 초기화
메뉴 > 측정 > 교정 초기화를 선택하면 모든 교정 데이터가 삭제됩니다.
- 초기화 후 교정 전 원시값이 출력됩니다
- 다시 교정을 수행할 때까지 정확도가 보장되지 않습니다

## 주의사항
1. 센서 표면에 기포가 없도록 확인하세요
2. 기준 시료의 온도가 현장과 유사한 것이 좋습니다
3. 기준 시료의 농도는 실험실 분석값을 사용하세요
4. 교정 시 충분한 안정화 시간을 확보하세요
