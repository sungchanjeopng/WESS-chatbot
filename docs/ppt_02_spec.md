# ENV200 제품 규격

## 제품 구성
- **E200-C**: 외벽부착형 (Clamp-On) - 컨트롤러 + Clamp-On 센서
- **E200-S**: 배관형 (Spool-Piece) - 컨트롤러 + Spool-Piece 센서

## 기본 아이템

### 컨트롤러 (C2S-210)
- 센서에서 수신한 초음파 신호를 분석하여 농도를 산출하는 본체

### 센서
| 센서 | 타입 | 설명 |
|------|------|------|
| S2C | Clamp-On | 외벽부착형, 비접촉식 |
| S2S | Spool-Piece | 배관삽입형, 접촉식 |

## 옵션 아이템

### 설치 구조물
| 옵션 | 설명 |
|------|------|
| ICM | 설치 클램프 |
| IPX | 방수 보호 구조물 |

## 주문 코드
- 기본 아이템: 컨트롤러 + 센서 + 제품 조합
- 옵션 아이템: 설치 구조물(ICM, IPX)
