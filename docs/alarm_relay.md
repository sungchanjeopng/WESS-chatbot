# ENV200 (C2S-211) 알람/릴레이 설정 가이드

## 릴레이 사양
| 항목 | 사양 |
|------|------|
| 릴레이 수 | 1채널 |
| 동작 방식 | 상한 알람 (농도가 동작값 이상이면 ON) |

## 릴레이 동작 원리

릴레이는 측정된 농도값을 기준으로 동작합니다:
- **농도 ≥ 동작값 (Act)** → 릴레이 ON
- **농도 ≤ 정지값 (Off)** → 릴레이 OFF

동작값과 정지값 사이에 차이를 두어 릴레이가 반복적으로 ON/OFF 되는 것(채터링)을 방지합니다.

## 설정 파라미터

### 일반 모드
| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| 동작값 (Act) | 0 이상 | 800 | 릴레이 ON 기준 농도 |
| 정지값 (Off) | 0 이상 | 700 | 릴레이 OFF 기준 농도 |

### 플랜트 모드
| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| 동작값 (Act) | 1000~9999 | 1150 | 릴레이 ON 기준 농도 |
| 정지값 (Off) | 1000~9999 | 1140 | 릴레이 OFF 기준 농도 |

## 릴레이 테스트
메뉴 > 출력 > 릴레이 테스트에서 릴레이를 수동으로 ON/OFF 할 수 있습니다.

## 설정 예시
농도가 800 이상이면 알람, 700 이하면 해제:
- 동작값: 800
- 정지값: 700
- 농도가 750일 때: 이전 상태 유지 (채터링 방지 구간)
