# ENV200 (C2S-211) 알람/릴레이 설정 가이드

## 릴레이 사양
| 항목 | 사양 |
|------|------|
| 릴레이 수 | 1채널 |
| 동작 방식 | 상한 알람 (농도가 동작값 이상이면 ON) |

## 릴레이 동작 원리
- 농도 ≥ 동작값 (Act) → 릴레이 ON
- 농도 ≤ 정지값 (Off) → 릴레이 OFF

## 설정 파라미터
### 일반 모드
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 동작값 (Act) | 0 이상 | 800 |
| 정지값 (Off) | 0 이상 | 700 |

### 플랜트 모드
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 동작값 (Act) | 1000~9999 | 1150 |
| 정지값 (Off) | 1000~9999 | 1140 |
