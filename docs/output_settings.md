# ENV200 (C2S-211) 4-20mA 출력 설정 가이드

## 출력 사양
| 항목 | 사양 |
|------|------|
| 출력 타입 | 4-20mA 전류 출력 |
| 출력 채널 | 1채널 |

## 출력 범위 설정
- 4mA 설정값: 일반 기본 0, 플랜트 기본 1000
- 20mA 설정값: 일반 기본 800, 플랜트 기본 1200

### 출력 계산
출력(mA) = 4 + (측정값 - 4mA설정값) / (20mA설정값 - 4mA설정값) x 16

## 출력 트림 (미세 조정)
| 파라미터 | 범위 | 기본값 |
|----------|------|--------|
| 12mA 트림 | 0~99 | 50 |
| 20mA 트림 | 0~99 | 50 |

## 에러 시 출력 동작
| 모드 | 동작 |
|------|------|
| 유지 (Hold) | 에러 직전 출력값 유지 |
| 3.8mA | 하한 경보 전류 |
| 21mA | 상한 경보 전류 |

## 외부 입력 연동
- 외부 입력 1/2 신호를 통해 세정장치 등과 연동 가능
- 시작/종료 대기 시간: 0~3600초 (기본: 30초)
