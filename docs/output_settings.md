# ENV200 (C2S-211) 4-20mA 출력 설정 가이드

## 출력 사양
| 항목 | 사양 |
|------|------|
| 출력 타입 | 4-20mA 전류 출력 |
| 출력 채널 | 1채널 |

## 출력 범위 설정

### 4mA 설정값 (Range Low)
- 4mA에 대응하는 측정값
- 일반 모드 기본값: 0
- 플랜트 모드 기본값: 1000

### 20mA 설정값 (Range High)
- 20mA에 대응하는 측정값
- 일반 모드 기본값: 800
- 플랜트 모드 기본값: 1200

### 출력 계산
출력(mA) = 4 + (측정값 - 4mA설정값) / (20mA설정값 - 4mA설정값) x 16
- 측정값이 4mA설정값 이하이면 4mA 출력
- 측정값이 20mA설정값 이상이면 20mA 출력

## 출력 트림 (미세 조정)
| 파라미터 | 범위 | 기본값 | 설명 |
|----------|------|--------|------|
| 12mA 트림 | 0~99 | 50 | 중간점 미세 조정 |
| 20mA 트림 | 0~99 | 50 | 상한점 미세 조정 |

트림값 50이 기준(보정 없음)이며, 50보다 크면 출력 증가, 작으면 출력 감소.

## 에러 시 출력 동작
| 모드 | 동작 | 용도 |
|------|------|------|
| 유지 (Hold) | 에러 직전 출력값 유지 | 일시적 에러 시 출력 변동 방지 |
| 3.8mA | 3.8mA 출력 | 상위 시스템에 에러 알림 (하한) |
| 21mA | 21mA 출력 | 상위 시스템에 에러 알림 (상한) |

## 외부 입력 연동
- 외부 입력 1/2 신호를 통해 세정장치 등과 연동 가능
- 외부 입력 시 출력 동작: 유지 또는 4mA 선택
- 시작/종료 대기 시간: 0~3600초 (기본: 30초)

## 4mA 출력 테스트
메뉴 > 출력 > 4mA 출력에서 수동으로 4mA를 출력하여 루프 테스트 가능.
